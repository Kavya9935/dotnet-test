# AWS Transform Demo Projects

Two .NET Framework 4.7.2 projects ready for AWS Transform:

## Project 1: Web API
- Uses Newtonsoft.Json package
- Simple REST API structure
- Located in: Project1-WebApi/

## Project 2: Class Library
- Uses System.Data.SqlClient
- Database service layer
- Located in: Project2-ClassLibrary/

## Usage with AWS Transform:
1. Push this folder to your source control repository
2. Run AWS Transform and select the repository
3. AWS Transform will upgrade both projects from .NET Framework 4.7.2 to .NET 8.0
